# Music-Player
Музыкальный плеер на JavaScript

Верстка под desctop, адаптива нет
Только HTML, CSS и JS

API не используется. 
Данные описанные в data.js, треки берутся из папки audio
Скрипт полключается как module, запускать через LiveServer
